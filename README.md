# Devfolio-pachara

เว็บไซต์ Portfolio ที่พัฒนาด้วย React สำหรับใช้ประกอบการสอนในรายวิชาสัมมนา โดยมุ่งเน้นการแสดงตัวอย่างการพัฒนาเว็บสมัยใหม่ การออกแบบ UI/UX และการจัดโครงสร้างโปรเจกต์แบบมืออาชีพ พร้อมนำเสนอผลงานในรูปแบบ Web Application จริง

> โปรเจกต์นี้จัดทำขึ้นเพื่อการศึกษาและการสอนเท่านั้น

---

## Features

- แสดงข้อมูลโปรไฟล์และผลงาน (Portfolio Showcase)
- โครงสร้างโปรเจกต์แบบ React สมัยใหม่
- ออกแบบ UI/UX ให้เหมาะกับการใช้งานจริง
- รองรับ Responsive (Desktop / Tablet / Mobile)
- ใช้ Animation เพื่อเพิ่มประสบการณ์ผู้ใช้
- เหมาะสำหรับใช้เป็นตัวอย่างในการเรียนการสอน

---

## Tech Stack

- React
- Vite
- Tailwind CSS / CSS3
- Framer Motion (Animation)
- JavaScript (ES6+)
